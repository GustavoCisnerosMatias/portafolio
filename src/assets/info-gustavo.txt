1. CAC Santa Elena (Freelance)
   DESCRIPCIÓN GENERAL:
   Sistema web desarrollado para el Centro de Atención a la Comunidad (CAC) de Santa Elena, diseñado para optimizar completamente la gestión de asistencia institucional, eliminando el registro manual en hojas de Excel que era lento, propenso a errores y difícil de escalar.

   PROBLEMA RESUELTO:
   El CAC registraba la asistencia de forma manual en Excel, lo que generaba pérdida de tiempo, inconsistencias en los datos y dificultad para consultar el historial. Gustavo desarrolló un sistema web completo que digitalizó y automatizó este proceso.

   FUNCIONALIDADES IMPLEMENTADAS:
   - Registro de asistencia individual y múltiple (varios IDs en un solo envío), reduciendo el tiempo por operación.
   - Consulta de historial de asistencias por participante, fecha y actividad.
   - Panel administrativo para gestión de usuarios, actividades y reportes.
   - Importación masiva de datos desde Excel hacia la base de datos Firestore.

   SCRIPTS PYTHON DESARROLLADOS:
   - Script 1: Limpieza y preparación de datos existentes en Excel para asegurar integridad antes de la migración. Permitió importar masivamente más de 1.000 usuarios directamente al sistema sin intervención manual.
   - Script 2: Extracción de actividades pasadas e IDs de participantes desde registros históricos, habilitando la reconstrucción del historial completo de asistencias en el nuevo sistema.

   IMPACTO EN PRODUCCIÓN (evidenciado en Firestore):
   - El sistema generó un alto volumen de lecturas, escrituras y consultas en Firestore (NoSQL), demostrando adopción real y uso activo por parte de los usuarios del CAC.
   - Más de 1.000 usuarios migrados exitosamente desde Excel a Firebase Firestore mediante automatización Python.
   - Eliminación total del registro manual, reduciendo errores y tiempos de gestión.

   STACK TECNOLÓGICO: Angular + Firebase (Firestore NoSQL) + Python (scripting de migración y limpieza de datos)
